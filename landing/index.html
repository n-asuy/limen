    Set up once, keeps forever

    Names, emojis, and app history persist across restarts, saved locally as plain JSON.

    Two switches to flip.

    macOS has no public API for switching Spaces, so Limen drives the system the honest way: by pressing the keys macOS already understands. That needs two settings.

    Accessibility permission

    Limen synthesizes the switching key events, which macOS gates behind Accessibility. It prompts on first launch, and reopens the right pane if a switch ever fails.

    System Settings › Privacy & Security › Accessibility

    Desktop switching shortcuts

    macOS ships the Ctrl+1-9 desktop shortcuts disabled. Enable them once, and Limen's settings window shows you which of the nine are live.

    System Settings › Keyboard › Shortcuts › Mission Control

    No accounts, no analytics, no network. Everything Limen knows lives in ~/Library/Application Support/Limen and never leaves your machine.
